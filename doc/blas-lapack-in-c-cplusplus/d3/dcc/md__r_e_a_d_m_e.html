BLAS/LAPACK in C and C++
Demo of BLAS/Lapack in C/C++
Introduction

This repository contains little examples for using BLAS/Lapack in C/C++. It can be used to build a large library for numerical linear algebra.

BLAS

The header file cblas.h can be obtained via

wget http://www.netlib.org/blas/cblas.h

On macOS, we have to link to -lblas, which resolves to

$ otool -L test_blas
test_blas:
/System/Library/Frameworks/Accelerate.framework/Versions/A/Frameworks/vecLib.framework/Versions/A/libBLAS.dylib (compatibility version 1.0.0, current version 1.0.0)
/usr/lib/libc++.1.dylib (compatibility version 1.0.0, current version 120.1.0)
/usr/lib/libSystem.B.dylib (compatibility version 1.0.0, current version 1226.10.1)

Documentation

We use Doxygen for documentation generation.

To install doxygen on macOS, use

brew install doxygen

or install it from source.

To generate documentation for this repository, run

./run-doxygen.sh
