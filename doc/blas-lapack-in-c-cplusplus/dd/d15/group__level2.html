GEMV : y = alpha * A * x + beta * y.

A: M rows, N columns; M and N are independent of TransA.

Assume A is contiguous.

If order == CblasRowMajor,

  • lda == N, number of columns
  • when A is not contiguous, lda = N + column_padding

If order == CblasColMajor,

  • lda == M, number of rows
  • when A is not contiguous, lda = M + row_padding
Warning
The need for lda. For the vector X, we need incX so that we know how to get the next element. For the matrix A, we have to known how to jump to the next row (if it is column major) or to the next column (if it is row major). Since A may not be contiguous in memory, we have to provide lda.
