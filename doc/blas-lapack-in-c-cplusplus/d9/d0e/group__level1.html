Detailed Description

Level 1 functions in BLAS.

y = sum(abs(x[:]))
y = alpha * x + y
y = x
y = alpha * y
s = dot(x, y)
Note
x and y can be the same; that is, it can be an inplace operation.

Refer to https://en.wikipedia.org/wiki/Basic_Linear_Algebra_Subprograms#Level_1 for the classification of BLAS levels.

CBLAS_INDEX kk::level1::isamax ( const int  N,
const float *  X,
const int  incX 
)

Return the index i such that abs(X[i*incX]) is the maximum.

The naming convention of isamax

  • i: the return value is an integer
  • s: single float precision
  • amax: short for **a**bsolute max

Example: for an array [0, 1, 2, -3]

  • if N = 4, incX=1, then it returns 3 (zero based index, the largest absolute value 3 is at position 3, [0, 1, 2, -3])
  • if N = 2, incX=2, then it returns 1 (zero based index, the largest absolute value 2 is at position 1, with stride 2, [0, 2])
Parameters
Nnumber of elements in X
Xthe vector X
incXthe stride in X
Returns
the index i such that X[i*incX] is the maximum
Warning
it should be CBLAS_INDEX, which is platform dependent macro
