Detailed Description

template<typename Dtype, int N>
class cnn::Jet< Dtype, N >

Jet for autodiff (forward mode).

This class is based on https://github.com/kashif/ceres-solver/blob/master/include/ceres/jet.h

We use the same name Jet as in ceres-solver.
